# Подземелья Тьмы

Roguelike dungeon crawler в стиле NetHack. Открой `index.html` в браузере и играй.

## Управление

- **WASD / Стрелки** — движение
- **Пробел** — атака / взаимодействие / спуск по лестнице

## Возможности

- Процедурная генерация карты 50x50
- 6 видов врагов с нарастающей сложностью
- Система уровней и прокачки
- Зелья, оружие, щиты, сундуки с лутом
- Миникарта и fog of war
- Бесконечные этажи
